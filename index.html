Map va Set

JavaScriptda Map va Set obyektlari ko'pincha ma'lumotlarni tuzilmalash va qayta ishlash uchun ishlatiladi.Bu obyektlar, katta hajmdagi ma'lumotlarni samarali boshqarish va izlash uchun juda qulaydir.Map: Kalit-qiymat (key-value) juftlarini boshqaradi.Set: Takrorlanmas (unikal) qiymatlarni boshqaradi.

Map vs Set Types

Map methods set(key, value): Kalit-qiymat juftini qo'shadi yoki yangilaydi.
get(key): Kalitga mos qiymatni qaytaradi.
has(key): Kalit mavjudligini tekshiradi.
delete(key): Kalit-qiymat juftini o'chiradi.
clear(): Hammasini tozalaydi.
size: Elementlar sonini qaytaradi.

let map = new Map()
map.set( "a", 1 );
map.set( "b", 2 );
map.set( "c", 3 );

console.log(map.get("a")); // Natija: 1
console.log(map.has("b")); // Natija: true
console.log(map.size); // Natija: 3

map.delete("c");
console.log( map.size ) // Natija: 2 c o'chirildi

map.clear();
console.log( map.size ) // Natija: 0 barcha elemnt o'chirildi

Izoh set(key, value): Kalit-qiymat juftini qo'shadi (a: 1, b: 2, c: 3).
get(key): Kalit a uchun qiymatni qaytaradi (1).
has(key): Kalit b mavjudligini tekshiradi (true).
delete(key): Kalit cni o'chiradi, natijada hajmi 2 bo'ladi.
clear(): Barcha elementlarni o'chiradi, natijada hajmi 0 bo'ladi.

Set methods add(value): Qiymatni qo'shadi.
has(value): Qiymat mavjudligini tekshiradi.
delete(value): Qiymatni o'chiradi.
clear(): Hammasini tozalaydi.
size: Elementlar sonini qaytaradi.

let set = new Set()
set.add( 1 );
set.add( 2 );
set.add( 3 );
set.add( 1 ); // 1 allaqachon mavjud

console.log(set.has(1)); // Natija: true
console.log(set.has(4)); // Natija: false
console.log(set.size); // Natija: 3

set.delete(2);
console.log( set.size ) // Natija: 2

set.clear();
console.log( set.size ) // Natija: 0 barcha element o'chirildi

Izoh add(value): Qiymatlarni qo'shadi (1, 2, 3).
has(value): Qiymat 1 mavjudligini tekshiradi (true).
delete(value): Qiymat 2 ni o'chiradi, natijada hajmi 2 bo'ladi.
clear(): Barcha elementlarni o'chiradi, natijada hajmi 0 bo'ladi.
